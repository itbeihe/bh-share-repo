# IT行业的工作流程
## 演讲者：唐琪

[slide]

工作流程是指企业内部发生的某项业务从起始到完成，由多个部门、多个岗位、经多个环节协调及顺序工作共同完成的完整过程。简单地讲，工作流程就是一组输入转化为输出的过程。

[slide]
“一个萝卜一个坑”，无论何时在某个岗位上出现了工作的失误都能迅速且准确地找到责任人，这样可以有效地防止相关工作的不同岗位间的互相扯皮、踢皮球的现象。

[slide]

##工作流程的目的
* 明确职责
* 保证质量
* 提升效率

[slide]
